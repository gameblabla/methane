======================
Super Methane Brothers
======================
TI Nspire port by gameblabla

Super Methane Brothers is an Amiga game for developped by Apache Software Ltd.
It's very similar to Bubble Bobble.

==============
INSTALLATION
==============

Super Methane Brothers requires Ndless 3.1+.
Simply put methane.tns somewhere on your calc.
To start the game, launch methane.tns.
